關於我

這是我的第一個作品集網站，以作品集本身即為作品創作，靈感來源自大荒北經中的燭龍，以其幽玄之眼照看山海世界。

作品／

北經殘卷

此作品集以呈現平台概念製作，佐以甲蟲圖繪綴飾視覺風格。不適應北經殘卷者，可參此作品集。

作品／

羽衣悔換

這是我第一個完整練習網頁，主要熟學習網頁佈局以及引用第三方套件製作簡易動態效果，以及使用 Java Script 調動元素。

作品／

餐廳網站

透過練習製作此網站，加強CSS 語言中的位置與網格練習，同時有系統的將顏色、文本字體、行距先行設置。

作品／

旅遊網站

影片播放作為網站背景蔚為潮流，透過此渡假網站做為學習，並加入點擊切換功能。同時練習全頁覆蓋的導航、會員登入的寫法。

作品／

渡假網站

陳列、平面設計，加上一些話題展覽與活動影片剪輯。這是我在新光三越的作品。
